# zhuch.github.io

Personal website source for GitHub Pages.

## Quick Start

1. Edit `index.html` with your real content (name, links, project descriptions).
2. Replace `hello@example.com` in `index.html` with your real email address.
3. Commit and push to the `main` branch.

## Enable GitHub Pages

1. Open your repository on GitHub.
2. Go to **Settings** -> **Pages**.
3. Under **Build and deployment**, set:
	- **Source**: Deploy from a branch
	- **Branch**: `main` and `/ (root)`
4. Save and wait for deployment.

Your site URL will be:

`https://<your-github-username>.github.io/`

## Files

- `index.html`: Homepage markup and content.
- `styles.css`: Visual design, animations, and responsive layout.

## Customize Further

- Add more sections in `index.html` (blog, resume, timeline).
- Swap the color variables at the top of `styles.css`.
- Add a custom domain later using a `CNAME` file.
